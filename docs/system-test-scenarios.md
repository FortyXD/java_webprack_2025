# Сценарии системного тестирования

Системные тесты находятся в `system-tests/test_bank_ui.py` и запускаются через `ant system-test`.
Тесты предполагают, что приложение уже поднято и доступно по `WEBTEST_BASE_URL`
или по значению Ant-свойства `webtest.baseUrl`.

## Сценарии

1. Главная страница и фильтр отделений:
   проверяется загрузка сводки, переход в раздел отделений и поиск отделения.

2. Отделения:
   проверяется ошибка создания без названия, успешное создание нового отделения,
   а также запрет удаления отделения, к которому прикреплены клиенты.

3. Клиенты:
   проверяется ошибка создания без отделения, успешное создание клиента,
   а также запрет удаления клиента с открытыми счетами или ненулевым балансом.

4. Счета и операции:
   проверяется успешное открытие накопительного счета, успешное закрытие счета
   с нулевым балансом, отказ в списании при недостатке средств, успешное
   начисление и отказ закрыть счет с ненулевым балансом.
